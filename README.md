## ğŸš€ Getting Started

### Prerequisites

- Node.js >= 18.0.0
- npm or yarn

### Installation

1. Clone the repository

```bash
git clone https://github.com/ecell-kccemsr/Ecell--2025-26.git
cd Ecell--2025-26
```

2. Install frontend dependencies

```bash
cd frontend
npm install
```

3. Start the development server

```bash
npm run dev
```

The site will be available at `http://localhost:5173`
